# To do list - SAPUI5 Bootcamp

## Application Information
A simple to do list project for SAPUI5 Bootcamp for 2018.

### Key Features
* View to do list items

## Deployment setup
* Run locally or
* Deploy to ABAP repository using SAP Web IDE.
